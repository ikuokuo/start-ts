## Installing TypeScript

```bash
npm install -g typescript

# npm outdated -g --depth=0
npm update -g typescript
```

## Running your TypeScript web app

```bash
make TS=handbook/typescript-in-5-minutes/1_greeter.ts HTML=1
```
